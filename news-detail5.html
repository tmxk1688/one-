天马行空创意 AI 软件正式发布！Android 版多模态创作 App 上线

📅 2026-07-07 👁️ 阅读：0 📝 来源：天马行空创意团队
重磅消息：由天马行空创意团队自主研发的「天马行空创意 AI」Android 客户端正式上线！App 集成智能对话、AI 图片、视频、音乐生成与提示词助手，无需自行配置 API，注册即可体验。

7 月 7 日，天马行空创意团队正式对外发布「天马行空创意 AI」手机客户端。作为团队 AI 战略的核心产品，App 将网页端 AI 创作能力完整迁移至移动端，并针对手机场景做了交互与性能优化，让创作者随时随地释放灵感。

一、核心功能一览

  • 智能对话：支持剧本、代码、文档、通用问答等多场景创作模式；
  • AI 图片：文生图、多尺寸输出、模板与图库管理；
  • AI 视频 / 音乐：一键生成短视频素材与背景音乐；
  • 提示词助手：翻译、润色、优化、反推，提升提示词质量；
  • 每日签到：登录领积分，透明消耗，开箱即用。

二、零配置 · 官方正版 · 安全下载

App 用户无需自行申请或填写 API Key，注册登录即可使用。官网网页版 AI 创作中心则需自行配置 API Key 与 API URL。请认准官方域名 tmxkcy.xyz 及下载页，勿通过第三方网盘或「破解版」渠道安装，以保障账号与隐私安全。

三、网页版 vs App 版

  • 网页版（script-writing.html）：功能全面，适合大屏深度创作，但需自行配置 API Key 与 API URL（如 361API）；
  • App 版（Android）：无需配置 API，注册登录即用，适合移动场景与快速生成。

两者可配合使用：Web 端深度编辑，App 端随时创作。

四、与 Web 版协同，创作不断档

天马行空创意 AI App 与官网 AI 创作中心形成互补：Web 端适合深度编辑与大屏创作（需自备 API），App 端适合移动场景下的快速生成（免配置 API）。团队将持续迭代，后续将公布 iOS 版本动态。
