AI创作中心使用手册

1. 简介

AI创作中心是一个专业的AI创作工具，支持多场景编辑、格式设置、保存和分享功能。通过集成多种AI模型，您可以轻松生成对话、图片、视频和音乐等内容。

2. 快速开始

  1. 点击页面上方的⚙️ API配置按钮，输入您的API Key和API URL
  2. 选择您想要使用的AI模型
  3. 点击保存配置按钮
  4. 选择您想要使用的功能选项卡（对话、图片、视频或音乐）
  5. 填写相关参数并点击生成按钮
  6. 等待生成结果并查看

图2.1 API配置界面

3. 功能详解

3.1 对话功能

对话功能允许您与AI进行自然语言交互，获取回答、建议或创意内容。

  1. 在输入框中输入您的问题或指令
  2. 点击发送按钮或按Enter键
  3. 等待AI回复
  4. 查看格式化后的AI回复，支持代码块、链接、标题等格式

图3.1 对话功能界面

3.2 图片生成功能

图片生成功能允许您根据文字描述生成高质量的图片。

  1. 在图片描述框中输入详细的图片描述
  2. （可选）在参考图片框中输入图片URL或Base64编码作为参考
  3. 选择图片尺寸
  4. 选择图片生成模型
  5. 点击生成图片按钮
  6. 等待图片生成完成
  7. 查看生成的图片

图3.2 图片生成功能界面

如何使用参考图片：

使用图片URL：
  1. 找到您想要作为参考的图片
  2. 右键点击图片，选择"复制图片地址"或"复制图片链接"
  3. 将复制的URL粘贴到参考图片输入框中
使用Base64编码：
  1. 找到您想要作为参考的图片
  2. 使用在线Base64编码工具（如 base64-image.de）将图片转换为Base64编码
  3. 复制生成的Base64编码字符串
  4. 将复制的Base64编码粘贴到参考图片输入框中

图3.3 Base64编码工具界面

提示：Base64编码的图片会包含"data:image/xxx;base64,"前缀，请确保完整复制包括前缀在内的整个编码字符串。

3.3 视频生成功能

视频生成功能允许您根据文字描述或图片生成视频。

  1. 选择生成模式（图生视频或文字生成视频）
  2. 在视频描述框中输入详细的视频描述
  3. 如果选择图生视频模式，输入首帧图像的URL
  4. 选择视频模型、时长、分辨率、比例、运动幅度、音频和错峰模式
  5. 点击生成视频按钮
  6. 等待视频生成任务创建
  7. 使用任务ID查询视频生成进度
  8. 查看生成的视频

图3.4 视频生成功能界面

3.4 音乐生成功能

音乐生成功能允许您根据文字描述生成音乐。

  1. 在音乐描述框中输入详细的音乐描述
  2. （可选）在参考音乐框中输入音乐URL作为参考
  3. 选择生成模式、音乐时长、音乐模型、音乐类型和音乐情绪
  4. 点击生成音乐按钮
  5. 等待音乐生成任务创建
  6. 使用任务ID查询音乐生成进度
  7. 查看生成的音乐

图3.5 音乐生成功能界面

4. API配置

要使用AI创作中心的功能，您需要配置API Key和API URL。

  1. 点击页面上方的⚙️ API配置按钮
  2. 在API Key输入框中输入您的361API密钥
  3. 在API URL输入框中输入API基础URL，例如：https://www.361api.com
  4. 选择您想要使用的AI模型
  5. 如果选择自定义模型，输入自定义模型名称
  6. 点击保存配置按钮
  7. （可选）点击查看API余额按钮，查看您的API余额

重要提示：API配置信息只会保存在您的本地浏览器中，不会上传到任何服务器。请确保妥善保管您的API密钥，不要分享给他人。

提示：您可以通过 361api.com 注册并获取API密钥。

5. 常见问题

  • 问题：API请求失败，显示429错误
    解答：这是由于API速率限制导致的，请等待一段时间后再尝试，或减少请求频率。
  • 问题：API请求失败，显示500错误
    解答：这是由于API服务器内部错误导致的，请稍后再尝试。
  • 问题：无法在本地文件系统中运行
    解答：请使用本地服务器运行网站，例如使用Python的http.server或Node.js的http-server。
  • 问题：图片生成失败，显示无可用渠道
    解答：这是由于所选模型在当前API分组下不可用，请尝试使用其他模型。
  • 问题：API余额不足
    解答：请登录361API官方网站充值API余额。

6. 最佳实践

  • 对话功能：使用清晰、具体的问题或指令，提供足够的上下文信息。
  • 图片生成：使用详细、具体的描述，包括主题、风格、颜色、构图等要素。
  • 视频生成：使用详细的场景描述，包括动作、环境、氛围等要素。
  • 音乐生成：使用详细的音乐风格描述，包括节奏、乐器、情绪等要素。
  • API使用：合理使用API配额，避免频繁发送大量请求。
  • 内容审核：遵守API提供商的内容政策，避免生成不适当的内容。
