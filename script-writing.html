AI 创作中心使用手册

天马行空创意网 · 网页版 AI 创作中心集成对话、图片、视频、音乐与提示词工具。网页版需自行配置 API Key 与 API URL后方可使用；若不想配置 API，请下载 天马行空创意 AI App，注册登录即可体验。

1. 简介

AI 创作中心面向剧本与创意工作流，主要能力包括：

  • AI 对话：多轮对话、创作模式（通用 / 剧本 / 网页 / 代码 / Word / 图片 / PPT）、导出与历史记录
  • 图片生成：在对话「图片」模式中生成，支持 GPT Image 2 等模型，自动添加暗水印与「天马行空创意网 AI」标识
  • 视频生成：基于 PixVerse（pixverse-video），支持文生、图生、首尾帧、多参考、延长、模板等类型；预览区叠加暗水印
  • 音乐生成：文字描述生成音乐，异步任务查询
  • 提示词助手：翻译、润色、专业优化、图片反推提示词
顶部选项卡：对话、生成视频、生成音乐、使用手册。图片生成已并入「对话」创作模式，不再有独立图片 Tab。

2. 快速开始（网页版）

三种使用方式：① 百言辰月（推荐，内置 Token 免配置）· ② 自有 API（361API 等，全功能）· ③ App 免配置（下载 App，注册即用）

  1. 使用 本地 HTTP 服务 打开本站（勿直接用 file://，否则无法调用 API）。例如在项目目录执行：python -m http.server 8080 或 npm start
  2. 在顶部选择 ① 百言辰月 即可直接对话；或选 ② 自有 API 填写 Key 与 URL（推荐 https://www.361api.com）
  3. 进入对应功能 Tab，填写描述与参数后点击生成
  4. 视频 / 音乐为异步任务：创建后使用返回的 任务 ID 查询状态，完成后可播放、下载或分享

API 密钥仅保存在本机浏览器 localStorage，不会上传至本站服务器。请妥善保管密钥。

3. 功能详解

3.1 AI 对话

左侧为历史对话列表，可新建、切换会话；主区域输入问题后发送，支持 Markdown、代码块与图片展示。

团队自训练模型 · 百言辰月 0.5B：选择 ① 百言辰月 即可直接使用（已内置团队 ModelScope Token，免配置）。创空间：百言辰月 · BainMoon_chat。支持对话、识图、文档、剧本、PPT/Word。

创作模式（侧栏或输入区上方芯片）：

模式说明
通用日常问答与创意讨论
剧本微电影/短片/分镜/多集连载；角色卡、分镜表工作台（AI 生成 + 手动编辑）
图片根据描述生成图片（见 3.2）
写网页生成完整 HTML 页面，可导出为网页文件
代码生成输出可运行代码，支持单文件下载
Word 文档模拟 Word 纸页实时编写，完成后可导出 .doc / .pdf
PPT 演示AI 生成 JSON 结构 → 大纲预览 → 编辑器（10 套主题模板）→ 导出 PPTX / PDF / HTML

顶栏操作：导出整段对话为 Word / 最近 AI 回复为 PDF / 网页、清空当前对话、沉浸全屏（千问 / 豆包风格：浅色侧栏、气泡对话、推荐提问、悬浮输入框；Esc 退出，偏好会记住）。

PPT 演示模式：发送后 AI 返回结构化 JSON，自动弹出大纲预览；确认后进入 PPT 编辑器。生成时选「AI 自动设计」可让 AI 定制配色；编辑器内也可点击「AI 生成模板」仅重新设计主题（不改幻灯片内容）。顶部可切换内置主题或已保存模板，并导出 PPTX（16:9 · 主题配色）、PDF 或 HTML 演示稿。

剧本工作台：剧本模式生成后自动解析角色卡与分镜表；点击「剧本工作台」可手动编辑、AI 补充分镜/角色、管理多集连载，并导出 Word/CSV。

Word 文档模式：AI 回复会在对话区以 Word 纸页样式逐字写入；编写完成后，消息下方出现「导出 Word」「导出 PDF」按钮。PDF 优先使用浏览器下载；若库加载失败，会打开打印窗口，可选择「另存为 PDF」。

深度思考：开启后模型会进行更充分的推理（响应更慢、消耗更多额度）。Word 模式下会先显示「分析文档需求 → 规划章节 → 开始写入」等步骤。

单条消息：悬停用户或 AI 消息可导出该条为 Word / PDF（Word 模式）/ 网页 / 代码文件。

3.2 图片生成（对话 · 图片模式）

在创作模式中选择 图片，输入画面描述后发送即可生成。

  1. 选择画幅（1:1、16:9、9:16 等）与模型（推荐 GPT Image 2，见 361API 实验室）
  2. （可选）上传或粘贴参考图 URL / Base64，用于图生图 / 编辑（/v1/images/edits）
  3. 生成结果带暗水印与「天马行空创意网 AI」标识，并写入本地图库，避免对话历史过大
提示词助手 → 图片反推：上传图片或填 URL，由 Vision 模型反推中英/SD 等格式提示词，可一键填入图片生成。

3.3 视频生成（PixVerse）

使用模型 pixverse-video，接口为 POST /v1/video/create，查询为 POST /v1/video/query。详见 361API PixVerse 实验室。

生成类型用途需准备
文生视频纯文字生成动态画面创意描述
图生视频以首帧图驱动运动首帧图（上传 / URL / Base64）
首尾帧首帧到尾帧过渡首帧 + 尾帧各一张
多主体多参考图融合（1–7 张）多行 URL 或多图上传
延长视频在已完成任务上续生成原视频任务 ID
视频模板特效模板生成模板 ID（数字），可选 1 张参考图

参数说明：时长、分辨率（540p / 720p / 1080p）、画幅比例、运动幅度、是否带音频、是否错峰队列。提交后在下方结果区查看任务 ID，点击「查询状态」直至完成，支持下载与分享。

3.4 音乐生成

  1. 填写音乐描述；可选参考音乐 URL
  2. 选择生成模式、时长、模型、类型与情绪
  3. 点击生成，使用任务 ID 查询进度并播放结果

3.5 提示词助手

在对话侧栏点击 提示词助手 打开弹窗：

  • 翻译：中英提示词互译
  • 润色扩写：丰富画面与细节描述
  • 专业优化：面向绘图/视频的专业提示词结构
  • 图片反推：从图片反推提示词，可应用到图片生成

4. 模型设置

  1. 在顶部或侧栏选择 ① 百言辰月 / ② 自有 API / ③ App 免配置
  2. 选择 ① 百言辰月：已内置团队 Token，无需填写密钥，直接对话
  3. ② 自有 API：API Key 在 361api.com 注册获取；API URL 填基础地址如 https://www.361api.com
  4. ③ App：下载 App，注册登录即可，无需填写密钥
  5. 保存后可查看 API 余额（自有 API 路径，若接口支持）
功能典型接口路径
对话/v1/chat/completions
图片文生图/v1/images/generations
图片编辑（参考图）/v1/images/edits
视频创建 / 查询/v1/video/create、/v1/video/query

5. 常见问题

  • 无法发送请求 / CORS 错误
    请通过 http://localhost 等本地服务器访问，不要双击 HTML 用浏览器直接打开。
  • 429 请求过于频繁
    系统会自动重试；请降低并发或稍后再试。
  • 500 服务器错误
    多为上游繁忙，稍后重试或更换模型。
  • 图片生成失败 / 无可用渠道
    检查分组是否支持该模型（如 gpt-image-2），或更换模型与画幅。
  • 视频任务长时间 processing
    视频生成耗时较长，请多次点击「查询状态」；错峰模式可能排队更久但更省额度。
  • 图生视频要求首帧
    图生、首尾帧、模板等类型需按界面提示上传或加载图片；URL 需可公网访问（部分接口不支持纯 Base64）。
  • 对话历史过大 / 保存失败
    图片已存本地图库，历史中仅保留引用；可定期清空旧对话或删除浏览器站点数据。
  • API 余额不足
    登录 361API 控制台充值。

6. 最佳实践

  • 对话：问题具体、带上下文；写网页/代码/Word 时说明风格与篇幅。
  • 图片：写清主体、构图、光线、风格；参考图与描述一致；善用提示词助手润色后再生成。
  • 视频：描述镜头运动与环境氛围；图生视频首帧尽量清晰；首尾帧两张图风格接近效果更好。
  • 音乐：标明节奏、乐器、情绪与场景（如「轻爵士，咖啡厅背景」）。
  • 合规：遵守平台内容政策，勿生成违法违规内容。
